Selecting Elements By ID

To select a single element on the page that has an ID (and only one element on a page can have a given ID at a time), this method exists. In the example below, the element/node on the page with the ID of "mySubmit" would be returned.

				
document.getElementById("mySubmit");

Selecting Elements By Tag Name

To select all elements on a page with a given HTML tag name as a collection (even if there are zero or one elements on the page with that tag), this method exists. In the example below, all of the paragraphs on the page would be returned as an HTMLCollection (works like an array).

Remember, for every element in the DOM, you can access its attributes using the dot operator (.) after selecting it. These attributes can be changed to new values, or you can access them to just see what their current value is. These are referred to as "getters" and "setters." A getter "gets" the value for you but doesn't change it, whereas a setter allows you to set a new value for that attribute.

				
document.getElementsByTagName("p");

Selecting Elements By Class Name

To select all elements on a page with a given class as a collection (even if there are zero or one elements on the page with that class), this method exists. In the example below, all of the elements on the page that had the active class would be returned as an HTMLCollection (works like an array).

				
document.getElementsByClassName("active");

Selecting Elements By CSS Selector (as a group)

If you're comfortable with CSS selectors, the two querySelector methods in JS are ideal for finding DOM elements. (They also work similarly to the selectors we use in jQuery, so practicing with these methods is a good idea). This method will return a NodeList (works like an array) of all elements on the page that match the selector.

				
document.querySelectorAll("ul li");

Changing DOM Node Content

With the innerHTML method, we can add both text and HTML tag content to a DOM node on our page. With the textContent method, we can add just plain text content to a DOM node on the page. For form elements like inputs, the text they contain is actually accessed using the "value" attribute.

				
document.querySelector("h1").textContent = "New H1 Text";
document.getElementById("id").innerHTML = "Hello!";
let btnText = document.getElementById("mySubmit").value;

Removing an Attribute

The removeAttribute() method allows you to completely remove an attribute from an element by passing in the attribute name. In the book, you'll usually see this used to remove the "hidden" attribute from an element.

				
document.getElementById("mySubmit").removeAttribute("name");
